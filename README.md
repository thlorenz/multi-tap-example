# multi-tap-example

Example of setting up a project to run multiple tap tests in one go or one by one.

## Installation

    npm install thlorenz/multi-tap-example

    autoreconf --install

    ./configure

## Program

### Make program

    cd src && make

## Run program

    ./src/multi-tap-main

## Tests

### Make tests

    cd test && make

### Run tests

    make check


## Make all

    make
